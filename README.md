# OpenAccessArticles

OpenAccessArticles is a Python package for finding and retrieving open-access research PDFs from legal open-access repositories.

## What This Package Is For

- Searching for scholarly articles.
- Resolving accessible PDF links when content is legally available.
- Downloading open-access PDFs for research workflows.

## Legal and Ethical Use

This package is intended for lawful use with open-access sources and repositories.
It should be used only in compliance with publisher terms, repository policies, and local laws.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python OpenAccessLibrary/OpenAccessLibrary.py --help
```

## Goal

Provide a simple interface for working with legally available open-access research PDFs.
